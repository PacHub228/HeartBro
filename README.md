# HeartBro

HeartBro — это плагин для Minecraft серверов Paper/Spigot (1.21.11), который реализует механику кражи сердец. Игроки получают или теряют максимальное здоровье в зависимости от их успехов в PvP-сражениях.

## Особенности

- **Кража сердец:** Получайте 1 сердце за убийство игрока, теряйте 1 сердце при смерти от рук игрока.
- **Честная игра:** Игроки не могут опуститься ниже 3 сердец. Если вы убьете кого-то с 3 сердцами, вы не получите сердце.
- **Сохранение данных:** Все данные хранятся в локальной базе данных SQLite (`hearts.db`).
- **Инструменты администратора:** Команды для добавления, установки или удаления сердец у любого игрока.
- **Автодополнение:** Полная поддержка аргументов команд через Tab.

## Команды

| Команда | Описание | Право (Permission) |
|---------|----------|--------------------|
| `/heartbro help` | Показать меню помощи | `heartbro.use` |
| `/heartbro version` | Показать версию плагина | `heartbro.use` |
| `/heartbro heart <игрок> <add/set/remove> <количество>` | Изменить количество сердец игрока | `heartbro.admin` |
| `/heartbro cleardatabase` | Сбросить данные всех игроков до стандартных (10 сердец) | `heartbro.admin` |

## Права (Permissions)

- `heartbro.use`: Позволяет использовать базовые команды (По умолчанию: у всех)
- `heartbro.admin`: Позволяет использовать административные команды (По умолчанию: у OP)

## Установка

1. Скачайте последний JAR-файл со страницы релизов.
2. Поместите JAR-файл в папку `plugins` вашего сервера.
3. Перезапустите сервер.
4. Настройте права (permissions) при необходимости.

## Сборка из исходников

Требования:
- Java 21
- Maven

```bash
mvn clean package
```
